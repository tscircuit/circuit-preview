TSCircuit TSX Script Conversion Test

This page demonstrates the automatic conversion of <script type="tscircuit-tsx"> to executable circuit code.

Circuit will be rendered below when processed...

Original TSX Script:

<script type="tscircuit-tsx">
  export default () => (
    <board width="10mm" height="10mm">
      <resistor name="R1" resistance="1k" footprint="0402" />
      <capacitor name="C1" capacitance="100nF" footprint="0402" />
    </board>
  )
</script>

How it works:

The script above gets automatically processed:

  1. JSX is extracted from the export pattern
  2. Babel transforms JSX to React.createElement calls
  3. A new Circuit instance is created and the component is added
  4. The circuit is rendered using CircuitJsonPreview

No manual setup required! Just include the scripts and use TSX syntax.
